## Doom Fire Algorithm Lua/Love2D Implementation

You should have Love2D framework installed in your system before run this code.

[Love2D](https://love2d.org/)

After love2d installed in your system, run love in your terminal and passing the folder name as a argument.

If you're using Powershell, it should be like: .\love.exe DoomFire

If you're using a Linux Terminal, it should be like: love DoomFire

[Working Video](https://www.youtube.com/watch?v=TNeFsGlwI8Q)
