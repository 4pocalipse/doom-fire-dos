### Embedded version of Doom Fire Algorithm

Implemented in C using SMT32F429 board and [mbed](https://os.mbed.com/users/agaelema/code/Doom_Flame-F429ZI_v02/).
* STM32F429 - Arm Cortex-M4 180MHz
